# 如何将 URL 转换为 JSON？

在 Web 开发和 API 测试中，经常会通过 URL query string 传递参数。将 URL 转换为 JSON 有助于快速理解 URL 中的信息，提升您的开发、测试效率。

<img src="/example/url-to-json.webp" alt="URL 转成 JSON" style={{ marginBottom: 10 }} />

通过 JSON For You 将 URL 转成 JSON 非常简单，只需按以下步骤操作：

1. 将带有查询参数的 URL 粘贴到左侧编辑器中；
2. 点击顶部的「搜索命令」，找到「**URL 转 JSON**」，点击执行。
