# 如何进行 JSON 格式化？

JSON For You 提供强大的 JSON 格式化（又叫 JSON 美化）功能，支持自动和手动两种方式，帮助开发者快速美化 JSON 数据，提升可读性。默认情况下，JSON For You 会在您**输入或粘贴 JSON 数据时自动进行格式化**。这意味着您无需额外操作，即可获得结构清晰、缩进整齐的 JSON 代码。

如果不需要自动格式化，想手动格式化，也可以按以下步骤操作：

1. 将原始 JSON 字符串粘贴到左侧编辑器中；
2. 点击侧边栏的「自动格式化」按钮将其关闭；
3. 点击顶部的「搜索命令」，找到「**格式化**」，点击执行手动格式化。

## JSON 格式化的应用场景

规范的 JSON 格式对于以下场景至关重要：

- **团队协作**：提高多人开发时的 JSON 可读性；
- **错误排查**：快速定位语法错误和结构问题；
- **API 开发**：确保请求/响应数据格式正确；
- **文档编写**：创建易于理解的 JSON 示例。
